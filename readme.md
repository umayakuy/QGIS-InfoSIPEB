# InfoSIPEB

**InfoSIPEB** es un complemento no oficial para **QGIS** desarrollado por **QGISBolivia.org**, que permite la conexión directa con la plataforma oficial **INFO-SIPEB** del Estado Plurinacional de Bolivia para consultar y cargar mapas temáticos de manera rápida y sencilla dentro de QGIS.

## Contenido del repositorio

Este repositorio incluye:

- 📂 **Código fuente** completo del complemento.
- 📦 **Instalador del complemento** (`.zip`) listo para instalar manualmente en QGIS.
- 📖 Documentación y recursos del proyecto.

## Instalación

### Desde el repositorio oficial de QGIS (si está disponible)

1. Abra **QGIS**.
2. Vaya a **Complementos → Administrar e instalar complementos**.
3. Busque **InfoSIPEB**.
4. Haga clic en **Instalar**.

### Instalación manual

Si el complemento **aún no se encuentra disponible en el Repositorio Oficial de Complementos de QGIS**, puede instalarlo manualmente:

1. Descargue el archivo **ZIP** disponible en este repositorio.
2. En QGIS vaya a:

   **Complementos → Administrar e instalar complementos → Instalar desde ZIP**

3. Seleccione el archivo descargado.
4. Haga clic en **Instalar complemento**.
